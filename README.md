# 📊 AI-Powered Business Intelligence Dashboard

An interactive Business Intelligence Dashboard built with Python and Streamlit, 
featuring an AI chatbot for natural language data querying.

## 🚀 Features
- Interactive sales and profit analysis
- Regional and category performance charts
- Monthly sales trend analysis
- Discount vs profit impact analysis
- AI chatbot to query data in plain English (powered by Groq)

## 🛠️ Tech Stack
- Python, Pandas, Plotly, Streamlit, Groq API

## 📦 Installation
pip install -r requirements.txt

## ▶️ Run
streamlit run dashboard.py
